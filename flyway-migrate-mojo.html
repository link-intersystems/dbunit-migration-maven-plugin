dbunit-migration:flyway-migrate

Full name:

com.link-intersystems.dbunit.maven:dbunit-migration-maven-plugin:1.0.4-SNAPSHOT:flyway-migrate

Description:

Migrates a collection of DBUnit data set files from one database schema version to another with the use of Flyway and testcontainers.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
